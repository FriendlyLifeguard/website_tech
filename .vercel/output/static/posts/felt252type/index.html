Introduction to Mod and Multiplicative Inverse

In mathematics, the modulo operation finds the remainder after division of one number by another (sometimes called modulus).

Here are the rules:

For a(modb)a \pmod ba(modb)

1. a < b then a mod b = a 
2. a = b then a mod b = 0 
3. a > b then a/b = r where r is the remainder of a/b

The multiplicative inverse is simply :

15∗x=115 * x = 115∗x=1

This x is the multiplicative inverse of 15.

So let say we want to find the value of x of the following equation

3∗x≡1(mod7)3 * x \equiv 1 \pmod{7}3∗x≡1(mod7)

The xxx is the multiplicative inverse of 333 and it is equivalent to 1(mod7)1 \pmod 71(mod7)

Our goal is to find the value of xxx s.t the value of (3∗x)÷7(3 * x) \div 7(3∗x)÷7 yields the same remainder as 1(mod7)1 \pmod 71(mod7).

Therefore, we can say that the value of xxx is 555 because 3∗5=153 * 5 = 153∗5=15 and 15÷7=215 \div 7 = 215÷7=2 with a remainder of 1 and it is equivalent to 1(mod7)1 \pmod 71(mod7).

Overview of Felt252 type in Cairo

Felt252 type is a special data type that exists in the Cairo language by Starknet. As from the name, it occupies 252 bits and is used to represent a number in the range of

#1 0<=x<P0 <= x < P0<=x<P

where P is a large prime number equal to

P=2251+17∗2192+1P = 2^{251} + 17 * 2^{192} + 1P=2251+17∗2192+1

In the case of an overflow issue, it uses(modP)\pmod P(modP) to bring the value back to within the range.
